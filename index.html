UI Controls Examples

1. Radio button

A UI control that allows the user to select only one option from a predefined group of choices.

2. Checkbox

A UI control that allows the user to select one or multiple options independently from a set.

3. Text input

A field that allows the user to enter and edit text data.


4. Tabs

A navigation element that organizes content into separate sections, only one visible at a time.

7. Link

A navigational element that redirects the user to another page, section, or resource.
